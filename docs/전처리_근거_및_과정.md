## 2. Listwise Deletion — 표본 확정 과정

다중 회귀분석 및 머신러닝 모형은 투입되는 모든 변수(독립, 조절, 통제, 종속)에 단 하나의 결측치라도 존재할 경우 해당 사례 전체를 분석에서 제외하는 **완전사례분석(Listwise Deletion)** 방식이 적용된다. 이에 따라 원본 1,502개 사업장을 대상으로 종속변수 결측, 이상치, 주요 독립변수의 무응답을 순차적으로 제거하여 분석 표본을 확정하였다.

### 2.1 단계별 제거 과정 (재검증 완료)

(`notebooks/01_전처리.ipynb` 셀 4 로직을 원본 데이터로 재실행한 결과)

| 단계 | 제거 기준 | 제거 사유 | 제거 수 | 잔여 |
|---|---|---|---|---|
| 시작 | — | 제10차 실태조사(건설업) 원자료 | — | **1,502** |
| **1** | Q27_3_1, Q27_3_2, Q27_3_3 모두 NaN | 사고 발생 여부 판단 불가 (종속변수 결측) | **16** | 1,486 |

**총 제거**: 127개 (원본 1,502개의 8.5%)

### 2.2 재검증 코드

```python
import pandas as pd
raw = pd.read_csv('data/제10차 산업안전보건 실태조사_raw data_건설업_230824.CSV', encoding='cp949')
df = raw.copy()

# 1단계: 종속변수 결측
mask = df[['Q27_3_1','Q27_3_2','Q27_3_3']].isna().all(axis=1)
df = df[~mask]  # -16 → 1486

# 2단계: 이상치
df = df[df['Q27_3_3'] != 30]  # -1 → 1485

# 3단계: Q6=9
df = df[df['Q6'] != 9]  # -21 → 1464

# 4단계: Q10 무응답
df = df[~df['Q10'].isin([4, 9])]
df = df[df['Q10_1'] != 9]  # -62 → 1402

# 5단계: Q14 결측
df = df[df['Q14'].notna()]  # -24 → 1378

# 6단계: Q9=9
df = df[df['Q9'] != 9]  # -3 → 1375
```

```python
def encode_safety_org(row):

    elif row['Q6'] == 2:
        if row['Q7_1'] == 1:
            return 1
        else:
            return 0
    return np.nan
```

| 원본 문항 | 내용 |
|---|---|
| Q6 | 안전보건 전담부서 유무 (1=있음, 2=없음) |

**코딩 규칙**
- **2 (최상)**: 전담부서 있음 (Q6=1)
- **1 (중간)**: 전담부서 없음 + 안전 담당 직원 있음 (Q6=2, Q7_1=1)
- **0 (없음)**: 전담부서 없음 + 안전 담당 직원 없음 (Q6=2, Q7_1=2)

#### 4.1.2 위원회수준 (3단계, 0~2)

(`notebooks/01_전처리.ipynb` 셀 9)

```python
def encode_committee(row):
    if row['Q10'] == 3:
        return 0
    elif row['Q10'] in [1, 2]:
        if row['Q10_1'] == 1:
            return 2
        elif row['Q10_1'] in [2, 3]:
            return 1
        else:
            return 1
    return np.nan
```

| 원본 문항 | 내용 |
|---|---|
| Q10 | 산업안전보건위원회 운영 여부 (1=운영, 2=운영하나 형식적, 3=미운영, 4=잘 모르겠음) |
| Q10_1 | 위원회 이행관리 수준 (1=지속적, 2=가끔, 3=형식적) |

**코딩 규칙**
- **2 (운영 + 지속 이행관리)**: Q10∈{1,2} & Q10_1=1
- **1 (운영하나 이행관리 미흡)**: Q10∈{1,2} & Q10_1∈{2,3}
- **0 (미운영)**: Q10=3

**주의**: Q10=4(잘 모르겠음)는 표본 확정 단계(2단계 4항)에서 이미 제거되었음.

## 7. 제외한 변수 — 사유

### 7.1 노동조합 (Q5)

**박천수(2025) 포함 여부**: 포함 (표 2, p.73 "여건 변수" 항목)

**본 연구 제외 사유**: 노동조합 변수를 투입할 경우 이진 로지스틱 회귀에서 **통계적 수렴 실패**가 발생한다(계수 약 9, 표준오차 매우 큼). 이는 이진 종속변수 구조와 음이항 회귀(박천수 2025)의 차이에 기인한 것으로 판단된다. 본 연구의 RQ가 "내부 안전 관리 × 외부 기관 조절효과"임을 고려할 때, 노동조합은 핵심 관심 변수가 아니므로 제외한다.

### 7.2 지역 (SQ3)

**박천수(2025) 포함 여부**: 포함 (6개 권역 더미)

**본 연구 제외 사유**: 본 연구 RQ가 "내부 안전 관리와 실질적 안전 행동이 산업재해 발생에 미치는 영향: 외부 기관의 조절효과"이므로, 지역 차이는 본 RQ의 관심 대상이 아니다.

### 7.3 보건사고 (Q27_4 계열)

**박천수(2025) 포함 여부**: 별도 종속변수로 분석 (표 9, p.84)

**본 연구 제외 사유**: 박천수(2025) 표 3(p.75)에 따르면 안전사고 최대값은 30건(평균 0.530)인 반면, 보건사고는 최대값 3건(평균 0.019)으로 발생 빈도가 극히 낮다. 본 표본(n=1,375)에서 보건사고 발생 사례가 44건에 불과하여 이진 로지스틱 회귀의 통계적 검정력이 부족하다고 판단하여 본 연구에서는 안전사고만 분석한다.

**시사점**: 박천수(2025)는 보건사고에 대해 근로감독이 유의한 보호 효과(b=-1.043, p<0.01)를 보인다고 보고하였다. 본 연구는 이 결과를 Discussion 한계 섹션에서 언급하며, 안전사고와 보건사고의 기제 차이에 대한 후속 연구를 제안한다.

## 9. 한계 및 알려진 데이터 이슈

본 연구는 전처리 과정에서 다음과 같은 데이터 이슈를 발견하였으며, 투명성을 위해 기록한다.

### 9.1 외국인비율 분포의 0 편중

외국인비율은 Q4_3(외국인 근로자 수) ÷ Q4_1 × 100으로 계산되며, 본 표본에서 **외국인비율=0인 사업장이 전체의 48.3% (664개)**를 차지한다. 이는 건설업 외국인 고용의 현실을 반영하는 것이나, 연속형 변수로 투입 시 분포의 편중이 모델 적합에 영향을 줄 수 있다.

**본 연구의 처리**: 원본 그대로 연속형 투입 (박천수 2025와 동일).

### 9.2 리커트 변수 4~5점 편중

독립 B의 리커트 변수 4개(교육훈련도움, 정리정돈상태, 작업중지권, 작업반장기여)는 모두 중앙값이 4점, 75%분위가 5점으로 **4~5점에 응답이 집중**되어 있다. 본 연구의 SHAP 기반 Tree 모델이 해당 변수들의 분산을 과소평가할 수 있는 특성의 원인이며, 이는 `docs/데이터분석_과정_및_근거.md` 8.3절에서 상세 논의된다.

**해결 방안**: LR과 SHAP의 방법론적 차이를 투명하게 명시하고, 방향성 일치(LR OR<1, SHAP 4점 이상 음수 전환)를 근거로 정리정돈의 보호 효과를 제시.

### 9.3 횡단면 자료의 인과 식별 한계

본 연구는 2021년 단일 시점 횡단면 자료를 사용하므로, 변수 간 시간적 선후 관계를 식별할 수 없다. 박천수(2025) 본문(p.91):
> "이 연구는 2021년 조사자료를 기준으로 분석하였으므로, 인과관계보다 상관관계로 보는 것이 합리적이다."

본 연구도 동일한 한계를 가지며, 특히 고용노동부 감독과 사고 발생 간의 관계는 감독 대상 선정의 비무작위성(강민철 2024 p.169-178 참조, `docs/참고논문_정리.md` 1.4절)을 고려하여 해석해야 한다.
