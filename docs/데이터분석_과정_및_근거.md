## 1. 분석 개요

### 1.1 연구 질문 (RQ)

> 건설현장의 내부 안전 관리와 실질적 안전 행동이 산업재해 발생에 미치는 영향: 외부 기관의 조절효과를 중심으로

### 1.2 분석 전략

본 연구는 다음 두 가지 접근을 이중으로 적용하였다.

**① 로지스틱 회귀 (설명 중심)**
- 박천수(2025)의 통제변수 구조를 채택하여 각 변수의 순효과(partial effect)와 상호작용을 식별
- 계층적 회귀로 변수 블록별 설명력 기여도 검증
- 관련 파일: `docs/참고논문_정리.md` — 박천수(2025) 재현 비교표

**② 머신러닝 + SHAP (예측 및 비선형 패턴 탐색)**
- XGBoost, Random Forest, Logistic Regression, LightGBM 4개 모델 비교
- 데이터 불균형(사고 발생 28.4%) 대응을 위해 SMOTENC 적용
- SHAP 기반 변수 기여도 및 임계값 분석
- Permutation Importance로 삼중 교차 검증

### 1.3 Scenario D 채택 배경

초기 분석(Scenario A)에서는 카테고리 통제변수(공사종류·기성률·공사금액·발주처)를 **연속형(1~N)**으로 투입하였다. 이 구조에서 도출된 **고용노동부감독 OR=1.334, p=0.041**(유의) 결과가 박천수(2025) 건설업 연구의 **근로감독 b=0.08 비유의**(『노동정책연구』25권 4호, p.87) 결과와 충돌함을 발견하였다.

원인 추적 결과, 기성공정률(1=10% 미만, 2=10~30%, …, 6=90% 이상)을 연속형으로 투입하면 "기성률이 한 구간 증가할수록 사고 오즈가 일정 배수 변한다"는 **선형성 가정**이 부과되는데, 건설업 기성률과 사고 발생의 실제 관계는 단조 증가가 아니라 U자형 또는 계단형(10% 미만 낮음 → 중간 상승 → 90% 이상 다시 증가)이다. 이 비단조 패턴을 연속형이 포착하지 못하여 발생한 잔차가 감독 변수의 계수를 부풀리는 것으로 확인되었다. 이 원인 탐색 과정은 `scripts/replicate_park.py`(박천수 11변수 재현)와 `scripts/build_extended_csv.py`(박천수 추가변수 생성)에 기록되어 있다.

이에 따라 박천수(2025)가 사용한 **카테고리 더미 구조**(공사종류 6더미, 기성률 5더미, 공사금액 2더미, 발주처 2더미, 현장특성 3더미)를 채택하고, 박천수가 투입한 **추가 통제변수 4개**(신체위험·차량위험·1인당 안전보건관리비·현장특성 더미)를 함께 포함하여 **Scenario D**로 전면 재분석하였다.

### 1.4 최종 데이터셋

- 파일: `data/전처리_최종_확장.csv`
- 표본 크기: **n = 1,375** (박천수 2025 동일)
- 사용 변수: **33개** (독립 8 + 조절 3 + 통제 22)
- 사고 발생률: **28.4% (391/1,375)**
- SMOTENC 연속형 변수: 외국인비율, 인당관리비

## 5. A/B 갈래 탐색 (박사님 지적 대응)

### 5.1 배경

박사님의 지적: "독립변수를 A 그룹(내부 관리)과 B 그룹(현장 안전 행동)으로 나누어 분석했는데, 조절변수 3개를 하나의 블록으로 넣어 계층적 회귀에서 블록 단위 LR test p-value가 하나만 나오는 것은 해석이 모호하다. A 블록과 B 블록 각각에 대한 조절변수의 영향을 분리 탐색해야 한다."

이에 대응하여 본 연구는 다음과 같이 16개 모형을 추가로 탐색하였다.

- **A 갈래 8모형**: 통제변수 + 독립 A(3개) + 조절변수 조합 2³=8개
- **B 갈래 8모형**: 통제변수 + 독립 B(5개) + 조절변수 조합 2³=8개

- 베스트 선정 기준: **AIC 최소**

| 검증 | 삭제 사유 |
|---|---|
| **FDR / Bonferroni 다중비교 보정** | 33개 변수 동시 검정에 대한 보정은 탐색적 분석 단계에서 과도. 본 연구의 관심 변수는 독립·조절 11개로 제한적이며, 핵심 발견(정리정돈 유의성)은 A/B 갈래 탐색(16모형)에서 이미 robust로 확인됨. 보정 결과 정리정돈 p가 한계 수준으로 이동(p=0.041 → FDR p≈0.137)하여 오히려 해석 혼란 유발. |

### 8.3 LR과 SHAP의 순위 차이 — 방법론적 설명

LR의 Model 4에서 정리정돈상태는 유일하게 유의한 독립변수(OR=0.793, p=0.041)이나, SHAP Feature Importance 순위에서는 26/33위이다. 이 차이는 두 방법이 서로 다른 질문에 답하기 때문에 발생한다.

| 질문 | LR | SHAP |
|---|---|---|
| **무엇을 측정하는가** | 다른 변수가 통제된 상태에서 해당 변수의 **순효과 크기** | 모델이 각 샘플의 예측에 해당 변수를 **얼마나 많이 사용했는가** |
| **통계적 의미** | 계수의 유의성 (p-value) | 변수별 평균 기여도 (mean \|SHAP\|) |
| **분산 의존성** | 독립 (계수는 분산과 무관) | 높음 (분산이 큰 변수가 분기 기준으로 자주 사용됨) |

**정리정돈상태의 특성**:
- 리커트 1~5 범위, 평균 4.22, 표준편차 0.76
- Test set 분포: 1점 1개, 3점 37개, 4점 122개, 5점 115개 (노트북 02, 셀 47)
- **분산이 작은 변수** — Tree 기반 모델이 분기 기준으로 활용할 정보량이 제한됨

**통제변수 상위의 특성**:
- 기성률_1: 0/1 이진이지만 OR=0.089(약 11배 차이)로 큰 효과
- 인당관리비: 백만원/인 단위의 연속 변수, 분산 큼
- 외국인비율: 0~100% 연속 변수, 분산 큼

즉 SHAP 상위는 "분기할 때 큰 정보 이득을 주는 변수"이고, LR에서 유의한 변수는 "다른 변수 통제 후 일관된 방향성이 있는 변수"이다. 두 방법은 모순이 아니라 다른 측면을 드러낸다.
